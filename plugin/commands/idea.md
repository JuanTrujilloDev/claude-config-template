# /idea

Transform a raw idea into a structured feature specification.

## Usage

```
/idea <brief idea>
```

## Workflow

1. **Capture** — core concept, problem, beneficiaries
2. **Explore** — ask clarifying questions; identify constraints
3. **Spawn `po-manager`** to draft a brief
4. **Pause for approval**
5. After approval, hand off to `pm` via `/plan` for decomposition
